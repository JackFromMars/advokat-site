# Сайт адвоката Левченко Н.В. — Специфікація

## Огляд

Багатосторінковий сайт адвоката з Чернівців. Повна переробка існуючого сайту advokat-profi.com.ua на основі проведеного аудиту (оцінка 38/100). Мета — сучасний, швидкий, SEO-оптимізований сайт з mobile-first підходом.

**Тестовий домен:** advokat.jackmars.com.ua
**Мова сайту:** українська (без мультимовності)

## Стек технологій

- **Фреймворк:** Next.js 14 (App Router, SSG)
- **Стилі:** Tailwind CSS
- **Мова:** TypeScript
- **Хостинг:** Vercel (безкоштовний тариф)
- **Репозиторій:** окремий GitHub-репо
- **Контент:** JSON-файли у коді (з готовністю до міграції на Headless CMS)

## Дизайн

**Стиль:** Bento grid + Glassmorphism
**Кольори:** темно-синій/сірий + білий, золоті акценти — підкреслює серйозність та професійність адвоката
**Підхід:** Mobile-first

**Breakpoints:**
- 375px — mobile (base)
- 768px — tablet
- 1280px — desktop

**Mobile-first вимоги:**
- Кнопки мінімум 44x44px, достатні відступи між елементами
- Bento-сітка: 1 колонка (mobile) → 2 (tablet) → 3-4 (desktop)
- Чат-віджет та CTA «Зателефонувати» завжди доступні
- Форми — мінімум полів (ім'я + телефон), великі інпути
- Бургер-меню на мобільній версії

## Сторінки послуг — шаблон

Кожна сторінка послуги використовує єдиний шаблон `ServicePage`:
- Hero з назвою послуги та коротким описом
- Перелік підпослуг з описами
- FAQ специфічний для послуги
- CTA-блок з формою
- Хлібні крихти (breadcrumbs)

## SEO

- **SSG** — всі сторінки попередньо відрендерені
- **Schema.org:** Attorney, LocalBusiness, FAQPage
- **Мета-теги та Open Graph** для кожної сторінки
- **Sitemap.xml + robots.txt** — автогенерація Next.js
- **ЧПУ** — транслітеровані українські назви
- Кожна сторінка послуги оптимізована під свій кластер ключових слів (напр. "адвокат сімейні справи Чернівці")

## Інтеграції

### Google Places API
- Відгуки завантажуються при білді (ISR, revalidate кожні 24 години)
- Потрібен Google Cloud API ключ та Place ID

### Telegram Bot API
- Заявки з форми надсилаються через API Route (серверлес-функція)
- Потрібен Telegram Bot Token та Chat ID

### Google Maps
- Embed iframe на головній та сторінці контактів

### Google Analytics / GTM
- Підключення через `@next/third-parties`
